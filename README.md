This project performs Exploratory Data Analysis  and basic statistical analysis on the Adult Census Income dataset (`adult.csv`).  
The goal is to understand key numerical features that influence income classification and workforce patterns.

---

# What This Project Does

The notebook:

- Loads the dataset (`adult.csv`) using Pandas
- Prints and checks:
  - First few rows (`head`)
  - Dataset info (`info`)
  - Column names
  - Missing values
  - Summary statistics (`describe`)
- Selects important numerical variables related to income/workforce behavior
- Generates **histograms + KDE plots** for each selected feature
- Calculates measures of central tendency:
  - Mean
  - Median
  - Mode
- Calculates measures of variability:
  - Range
  - Variance
  - Standard deviation
  - IQR (Interquartile Range)
  - CV (Coefficient of Variation)
- Includes a short justification for why the variables were chosen
